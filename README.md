## Email/SMS Spam Classifier 🔍

This is a Flask-based web application that classifies text messages (Email/SMS) as **Spam** or **Not Spam** using Natural Language Processing (NLP) and a machine learning model. The app uses a trained Multinomial Naive Bayes classifier and TF-IDF vectorizer for prediction. Users can input a message, and the app will analyze and display whether it's spam.

### 🚀 Features:
- Preprocessing with tokenization, stemming, and stopword removal
- TF-IDF vectorization
- Trained Spam Classifier (Naive Bayes)
- Simple and clean web UI using Flask and HTML/CSS
- Ready for deployment on Render or any cloud platform

### 🛠 Tech Stack:
- Python
- Flask
- NLTK
- Scikit-learn
- HTML/CSS

### 📦 Model Files:
- `model.pkl` – Trained classification model
- `vectorizer.pkl` – Trained TF-IDF vectorizer
